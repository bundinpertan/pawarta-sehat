# 🌿 PAWARTA SEHAT
### Informasi Kesehatan Berbasis Cuaca

Aplikasi web ringan untuk mendeteksi risiko penyakit musiman 
berdasarkan kondisi cuaca real-time di lokasi pengguna.

🔗 **Akses langsung:** https://bundinpertan.github.io/pawarta-sehat

---

## Fitur
- 📍 Deteksi lokasi otomatis (GPS)
- 🌡️ Data cuaca real-time (suhu, kelembaban, curah hujan)
- 🦟 Peringatan 5 penyakit musiman: DBD, ISPA, Leptospirosis, Diare, Heat Stroke
- ✅ Tips pencegahan praktis per penyakit
- 📱 Mobile-friendly, bisa disimpan sebagai bookmark

## Dasar Acuan Ilmiah
- Kementerian Kesehatan RI (Direktorat P2PM)
- Jurnal Ekologi Kesehatan — Litbang Kemkes
- BMKG
- International Journal of Environmental Research and Public Health (2022)

## Teknologi
- Single-file HTML + CSS + JavaScript
- OpenWeatherMap API (cuaca real-time)
- Nominatim / OpenStreetMap (reverse geocoding)

## Disclaimer
Informasi bersifat indikatif berbasis parameter cuaca.
Bukan pengganti diagnosis medis. Selalu konsultasikan 
ke Puskesmas atau tenaga kesehatan untuk penanganan lebih lanjut.
